# Supply Chain Correlation Analysis

This repository captures the Excel-style correlation study for OptimalFlow Logistics.

## Contents

- `correlation.csv` &mdash; Correlation matrix computed from supplier performance metrics.
- `heatmap.png` &mdash; Red-White-Green heatmap mirroring Excel conditional formatting.
- `supply_chain_metrics.csv` &mdash; Input dataset (79 procurement observations).

## Reproducing

1. Open `supply_chain_metrics.csv` in Excel.
2. Enable the **Analysis ToolPak** add-in.
3. Generate the correlation matrix via `Data → Data Analysis → Correlation`.
4. Apply the Red-White-Green color scale through `Home → Conditional Formatting`.

The provided assets mirror the deliverables from the "Visualizing Forecasts with Excel" workflow.
